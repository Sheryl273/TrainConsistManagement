🚆 Train Consist Management App

Train Consist Management App is a console-based Java application that simulates how a railway system manages a train’s consist — a collection of bogies attached to an engine. The app supports passenger bogies with seat capacity tracking, goods bogies with cargo type and safety constraints, and provides realistic railway operations for planning, sorting, filtering, and reporting.

Features
Manage Passenger Bogies: Sleeper, AC Chair, First Class
Track seat capacities dynamically
Manage Goods Bogies: Rectangular, Cylindrical with cargo types
Sort, filter, and group bogies using modern Java features
Demonstrates object-oriented design and Java Stream API
